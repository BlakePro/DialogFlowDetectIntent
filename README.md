## Start

<a name="install"></a>
**Install Dependencies**
```sh
npm i
```

<a name="create"></a>
**Create .env && paste /private/key.json**
```sh
echo $'PROJECT_ID = \nPROJECT_SESSION_ID = \nPROJECT_FILENAME =' > .env && mkdir private
```

<a name="run"></a>
**Run**
```sh
node index.js "insert phrase"
```
